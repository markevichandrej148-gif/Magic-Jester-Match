Privacy & Data Processing Policy

Last Updated: June 3, 2026

This privacy declaration applies exclusively to data infrastructure integrated within Magic Jester Match. We believe in providing platform operators and application users with full transparency regarding instrumentation analytics.

Marketing Analytics & Installation Attribution: The application incorporates the AppsFlyer SDK. This structural tool is used to monitor marketing efficiency, aggregate usage statistics, and evaluate installation performance data.

App Tracking Transparency (ATT): For deployments inside compatible iOS matrix layers, the application requests operational consent via the native platform App Tracking Transparency framework. This mechanism facilitates metadata monitoring strictly to manage user acquisition attribution correctly.

Secure Internal Connections: The configuration parameters within the mobile client include dedicated functions capable of opening containerized views via highly secure **HTTPS WebView** protocols to reference terms, parameters, or documentation sheets.

Sandbox Save Profile: All game completion indicators, puzzle scores, active level progressions, and customized parameters are initialized and processed entirely on the local client device storage container. No global account profile is required to play the core matching content.
